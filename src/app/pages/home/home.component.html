Angular
¿Qué es?

Angular es un framework de desarrollo web basado en TypeScript. Permite crear aplicaciones web robustas y escalables.

Bootstrap
¿Qué es?

Bootstrap es un framework CSS que facilita la creación de interfaces web responsivas y atractivas

APIs
¿Qué son?

Las APIs (Interfaces de Programación de Aplicaciones) permiten la comunicación entre diferentes aplicaciones y servicios.
